# 正则
## 正则表达式
字符串以字母开头，后面字母数字下划线，长度6-30
```js
const reg = /^[a-zA-Z]\w{5, 29}$/
// ^ -- 字符串的开头
// [] -- 选择，多选的方式
// \w -- 字母、数字、下划线
// $ -- 结尾

// 一些简单的正则表达式
// 邮政编码
/\d{6}/
// 小写英文字母
/^[a-z]+$/
// 英文字母
/^[a-zA-Z]+$/
// 日期格式
/^\d{4}-\d{1, 2}-\d{1, 2}$/
// 用户名
/^[a-zA-Z]\w{5, 17}$/
// 简单的 IP 地址匹配
/\d+\.\d+\.\d+\.\d+/

// 正则规则
```
